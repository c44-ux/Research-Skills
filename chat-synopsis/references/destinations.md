## Per-destination notes

- **obsidian** — Must be an absolute path to a folder that already exists (or the user confirms creating it).
- **local** — Default root: `~/chat-synopses/` (Windows: `%USERPROFILE%\chat-synopses\`).
- **project** — Relative to workspace root; good for team repos.
- **research_playbook_folder** — Where **research-synopsis** / research mode saves playbooks. Can differ from general synopses folder. See [research-playbook.md](research-playbook.md).
- **notion** / **drive** — Requires the corresponding MCP server; fall back to file or chat if missing.
- **chat_only** — No file write; full markdown in the assistant message only.

## Per-run overrides

Triggers like `synopsis this chat → clipboard`, `synopsis this research → project`, or `→ research-playbooks` override defaults for one run only. Do not edit this file for overrides.
