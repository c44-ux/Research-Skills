# Research-Skills

Cursor skills and scripts for design, product, and customer research.

## Skills

| Skill | Folder | Summary |
|-------|--------|---------|
| **Evidence-based behaviour archetypes** | [`evidence-based-behaviour-archetype-creation/`](evidence-based-behaviour-archetype-creation/) | Build segment-level behaviour archetypes from **your** surveys and/or interviews; optional publish to **your** Miro via Miro MCP. |
| **Chat synopsis** | [`chat-synopsis/`](chat-synopsis/) | Capture a Cursor chat as a durable markdown synopsis (arc, attribution, standalone artifact)—not a transcript. Routes to Obsidian, local folder, or project path. |

See each skill’s `README.md` for install steps and usage.
