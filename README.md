# Research-Skills

Cursor skills and scripts for design, product, and customer research.

## Skills

| Skill | Folder | Summary |
|-------|--------|---------|
| **Evidence-based behaviour archetypes** | [`evidence-based-behaviour-archetype-creation/`](evidence-based-behaviour-archetype-creation/) | Build segment-level behaviour archetypes from **your** surveys and/or interviews; optional publish to **your** Miro via Miro MCP. |
| **Method synopsis** | [`method-synopsis/`](method-synopsis/) | Document rerunnable research methods from a Cursor session (arc, attribution, reproducibility playbook). Quant, qual, or mixed-methods. *Adapted from [vanessachang-dev/chat-synopsis](https://github.com/vanessachang-dev/chat-synopsis) (MIT); renamed and extended by Clare Reddan.* |

See each skill’s `README.md` for install steps, usage, and attribution.
